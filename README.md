# ClearSign

It is a Chrome browser extension which is designed to make online videos accessible to deaf community. Many videos, like on Youtube, aren't fully understandable for deaf users, even with captions. ClearSign solves this by using AI based avatar to show gestures based on video's audio.

# Features of ClearSign : 
1) Real-Time ASL Interpretation : Transcribes Youtube audio instantly to display American Sign Language (ASL) gestures, making videos understandable for deaf users.
2) Avatar : Shows clear ASL signs in a non-intrusive window, enhancing accessibility without relying on captions.
